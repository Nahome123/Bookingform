About my website




I have used the footer tag which is an HTML element that is used to create a footer section at the bottom of a web page. This tag is typically used to display copyright information, contact information, page navigation links, and other important information. I found the footer tag by searching for HTML elements on the web. The footer tag works by grouping together related information and providing a consistent look and feel to the bottom of a web page. This tag is nested inside the body tag, and can contain any other HTML elements such as headings, paragraphs, lists, or images.


I also used transform property in CSS which is a tool that allows developers to alter the way an element looks on the page. It can be used to rotate, scale, move, skew, and even perspective the elements. I found this property by doing some research on CSS properties and their uses. The way transform works is by taking a set of coordinates and manipulating them to create the desired effect. For example, a rotate transform would take a set of coordinates and rotate them around a central point to create a rotation effect. Similarly, a scale transform would take a set of coordinates and make them bigger or smaller to create a scaling effect. Transform is a powerful tool that developers can use to create dynamic and interesting effects on their websites.


I have used Nodemon which is a JavaScript library used to monitor for any changes in your source code and automatically restart your server. It is a great tool for development as it can save you time when you don't want to manually restart your server. I found Nodemon while doing research on how to speed up development and improve my workflow. It works by watching for file changes in your source code and then triggering a restart when a change is detected. It is a great tool for automatically restarting your server when changes are made, allowing for a faster development process.
